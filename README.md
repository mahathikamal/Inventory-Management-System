# Inventory Management System (Java)

A menu-driven Java application to manage product inventory using file handling and object-oriented concepts.

## Features
- Add new product
- Display all products
- Search product by ID
- Calculate total stock value

## Concepts Used
- Classes and Objects
- Constructors
- File Handling
- Exception Handling

## Learning Outcome
While doing my project, I learned how to build a Java program step by step. I also learned how to debug errors, test different cases, and handle user input properly.
